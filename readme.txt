This project is using Nodejs as Backend.
SQL database MYSQL and NoSQL is Redis.
Steps to install:
1. Clone the repo https://github.com/unknownboyy/Fortinet-Task.git
2. Inside the repo, run npm install ( it installs the all dependencies and modules required ).
3. Execute MySQL dump ( Login credentials are checked using MYSQL ).
4. Start Redis ( which stores session object ).
5. run node main.js ( wen-server ).
